# Tareas y archivo de ejecucion

Este directorio concentra:
- planes activos de trabajo
- historico de proyectos cerrados
- notas de ejecucion que no forman parte de la documentacion viva del sistema

## Convencion
- plan activo: `tasks/proyecto-[titulo].md`
- historico cerrado: `tasks/archive/`

## Regla
- `docs/` documenta el sistema
- `tasks/` documenta el trabajo realizado para llegar a ese sistema
