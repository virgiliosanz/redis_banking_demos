Cache-Aside Pattern

How it works

  1. Read request: Application checks Redis cache first.
  2. Cache hit: Data found in Redis → return immediately (<1ms).
  3. Cache miss: Data not in Redis → fetch from database (~200ms), store in Redis with TTL for next time.

Live Demo

Scenario: A banking product catalog. First fetch = cache miss (~200ms from DB). Second fetch = cache hit (<1ms from Redis). Clear cache to see the difference again.

Code Showcase

// ═══ Cache-Aside Pattern ═══
// 1. Check cache first (GET)
String cacheKey = "workshop:cache:product:" + productId;
String cached = redis.opsForValue().get(cacheKey);

if (cached != null) {
    // CACHE HIT — return immediately (<1ms)
    return deserialize(cached);
}

// 2. CACHE MISS — fetch from database (~200ms)
Map<String, Object> product = database.findById(productId);

// 3. Store in cache with TTL for next time
redis.opsForValue().set(
    cacheKey,
    serialize(product),
    Duration.ofSeconds(300)  // 5 min TTL
);

return product;

// ═══ Cache Eviction ═══
// On data update, invalidate the cache
redis.delete(cacheKey);

// ═══ Spring @Cacheable (alternative) ═══
@Cacheable(value = "products", key = "#productId")
public Product getProduct(String productId) {
    return database.findById(productId); // auto-cached
}
# ═══ Cache-Aside with Redis CLI ═══

# 1. Check cache (returns nil on miss)
GET workshop:cache:product:mortgage-fixed
# → (nil)  ← CACHE MISS

# 2. After fetching from DB, store in cache
SET workshop:cache:product:mortgage-fixed \
  '{"id":"mortgage-fixed","name":"Fixed Rate Mortgage"...}' \
  EX 300
# → OK  (TTL = 300 seconds = 5 minutes)

# 3. Next request — CACHE HIT!
GET workshop:cache:product:mortgage-fixed
# → '{"id":"mortgage-fixed","name":"Fixed Rate..."}'

# 4. Check remaining TTL
TTL workshop:cache:product:mortgage-fixed
# → (integer) 287

# 5. Evict on data change
DEL workshop:cache:product:mortgage-fixed
# → (integer) 1

# 6. Evict all cached products
# (use SCAN in production, KEYS for demo)
KEYS workshop:cache:product:*
DEL workshop:cache:product:mortgage-fixed \
    workshop:cache:product:savings-premium
