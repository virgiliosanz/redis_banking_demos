UC7

Feature Store (Risk Scoring)

Online feature store for real-time credit decisioning — versioned feature sets, Redis sub-ms fetches, and simulated ML inference.

How it works
  1. Batch credit features and real-time behavioural features are materialized into versioned Redis Hashes (`feature_set_v1`, `feature_set_v2`).
  2. When a new transaction arrives, the online feature set is refreshed with `HINCRBY` + `HSET` so inference always reads current behaviour.
  3. The inference API fetches the full feature vector with `HGETALL` in sub-millisecond latency and sends it to a simulated credit model.
  4. The UI shows the end-to-end pipeline plus a Redis vs database latency comparison for inline credit decisioning.

Code Showcase

String key = "uc7:feature_set_v2:client:C1001";

// 1. Online feature fetch from Redis Hash
long fetchStart = System.nanoTime();
Map<Object, Object> features = redis.opsForHash().entries(key);
double redisFetchMs = (System.nanoTime() - fetchStart) / 1_000_000.0;

// 2. Batch + real-time features served together
String paymentRatio = (String) features.get("batch_payment_ratio_90d");
String txRisk = (String) features.get("realtime_risk_score");

// 3. Simulated ML inference for credit decisioning
double approvalProbability = creditDecisionModel(features);
String decision = approvalProbability >= 0.72 ? "APPROVE"
        : approvalProbability <= 0.40 ? "DENY"
        : "REVIEW";

// 4. Keep online features fresh after each event
redis.opsForHash().increment(key, "realtime_tx_count_1h", 1);
redis.opsForHash().put(key, "realtime_risk_score", "42");

// 5. Compare with slower database fetch
double dbFetchMs = 150.0;
double savedMs = dbFetchMs - redisFetchMs;
# 1. Versioned feature sets in Redis
HGETALL uc7:feature_set_v1:client:C1001
HGETALL uc7:feature_set_v2:client:C1001

# 2. Online updates after a transaction event
HINCRBY uc7:feature_set_v2:client:C1001 \
  realtime_tx_count_1h 1
HSET uc7:feature_set_v2:client:C1001 \
  realtime_tx_amount_max_24h "850.00" \
  realtime_risk_score 42

# 3. Read the full feature vector for inference
HGETALL uc7:feature_set_v2:client:C1001
