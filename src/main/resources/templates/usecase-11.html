Real-time Transaction Monitoring

Live transaction metrics with Redis Streams — XADD, XRANGE, XLEN with manual aggregation.

How it works
  1. Ingest: Each transaction is written to a Redis Stream with XADD — amount and risk score as fields.
  2. Store: Redis Streams store timestamped entries with automatic IDs (ms-precision). XTRIM MAXLEN keeps the stream bounded.
  3. Query: XRANGE reads entries in a time window. The app aggregates into 1-second buckets (COUNT, AVG, SUM) for the dashboard.
  4. Detect: Anomaly bursts (spike in volume + high amounts) are instantly visible in the aggregated stream data.

Code Showcase

# ═══ Redis Streams — Transaction Monitoring ═══

# Add a transaction event to the stream
XADD workshop:txmonitor:stream * \
  amount 150.75 riskScore 25.3 \
  type purchase merchant "Store ABC"

# Read last 10 transactions
XRANGE workshop:txmonitor:stream - + COUNT 10

# Read transactions from last 60 seconds
# (replace <timestamp_ms> with epoch ms)
XRANGE workshop:txmonitor:stream <timestamp_ms>-0 + COUNT 100

# Count total transactions in the stream
XLEN workshop:txmonitor:stream

# Trim stream to last 10000 entries (retention)
XTRIM workshop:txmonitor:stream MAXLEN ~ 10000

# Read transactions in reverse (most recent first)
XREVRANGE workshop:txmonitor:stream + - COUNT 5
// ═══ Redis Streams with Spring Data Redis ═══

// Add a transaction to the stream
Map<String, String> txData = Map.of(
    "amount", String.valueOf(amount),
    "riskScore", String.valueOf(riskScore),
    "type", type,
    "merchant", merchant
);
redisTemplate.opsForStream().add(
    StreamRecords.string(txData)
        .withStreamKey("workshop:txmonitor:stream")
);

// Read transactions from last 60 seconds
long fromTs = System.currentTimeMillis() - 60000;
List<MapRecord<String, Object, Object>> records =
    redisTemplate.opsForStream().range(
        "workshop:txmonitor:stream",
        Range.closed(fromTs + "-0", "+")
    );

// Calculate metrics from stream data
long totalCount = redisTemplate.opsForStream()
    .size("workshop:txmonitor:stream");

// Aggregate: compute TPS, avg amount per second
Map<Long, List<Double>> buckets = new TreeMap<>();
for (var record : records) {
    long ts = Long.parseLong(
        record.getId().toString().split("-")[0]);
    long bucket = ts / 1000 * 1000; // 1-second buckets
    buckets.computeIfAbsent(bucket, k -> new ArrayList<>())
        .add(Double.parseDouble(
            record.getValue().get("amount").toString()));
}
