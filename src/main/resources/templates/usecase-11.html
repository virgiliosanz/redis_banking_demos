Real-time Transaction Monitoring

Live transaction metrics with Redis Streams — XADD, XRANGE, XLEN with manual aggregation.

How it works
  1. Ingest: Each transaction is written to a Redis Stream with XADD — amount and risk score as fields.
  2. Store: Redis Streams store timestamped entries with automatic IDs (ms-precision). XTRIM MAXLEN keeps the stream bounded.
  3. Query: XRANGE reads entries in a time window. The app aggregates into 1-second buckets (COUNT, AVG, SUM) for the dashboard.
  4. Detect: Anomaly bursts (spike in volume + high amounts) are instantly visible in the aggregated stream data.

Code Showcase

# ═══ Redis Streams — Transaction Monitoring ═══

# Add a transaction event to the stream
# The service emits only two fields: amount and riskScore
XADD uc11:stream:transactions * \
  amount 150.75 riskScore 25.3

# Trim stream right after XADD to keep it bounded
# (exact MAXLEN — no '~' flag in this service)
XTRIM uc11:stream:transactions MAXLEN 10000

# Read last 10 transactions
XRANGE uc11:stream:transactions - + COUNT 10

# Read transactions in a bounded time window
# (service uses <fromMs>-0 .. <toMs>-99999)
XRANGE uc11:stream:transactions \
  <fromMs>-0 <toMs>-99999 COUNT 10000

# Count total transactions in the stream
XLEN uc11:stream:transactions

# Read transactions in reverse (most recent first)
XREVRANGE uc11:stream:transactions + - COUNT 5
// ═══ Redis Streams with Spring Data Redis ═══

// Add a transaction — the service emits only amount + riskScore
Map<String, String> fields = new LinkedHashMap<>();
fields.put("amount",    String.format("%.2f", amount));
fields.put("riskScore", String.format("%.1f", riskScore));

MapRecord<String, String, String> record = StreamRecords
    .string(fields)
    .withStreamKey("uc11:stream:transactions");
redis.opsForStream().add(record);

// Keep the stream bounded — XTRIM MAXLEN (exact, no approximate flag)
redis.opsForStream().trim("uc11:stream:transactions", 10_000);

// Read a bounded time window — XRANGE key fromId toId COUNT N
long now = System.currentTimeMillis();
long fromMs = now - 60_000;
List<MapRecord<String, Object, Object>> records =
    redis.opsForStream().range(
        "uc11:stream:transactions",
        Range.closed(fromMs + "-0", now + "-99999"),
        Limit.limit().count(10_000)
    );

// Aggregate in Java into 1-second buckets (count, sum, max)
Map<Long, int[]> buckets = new TreeMap<>();
for (var r : records) {
    long ts = Long.parseLong(
        r.getId().getValue().split("-")[0]);
    long bucket = (ts / 1000) * 1000; // 1s buckets
    buckets.computeIfAbsent(bucket, k -> new int[]{0})[0]++;
}
